# 🛒 Amazon Homepage Clone

A responsive static clone of Amazon India's homepage built using HTML and CSS. Great for frontend learning and UI practice.

## 🌐 Features
- Responsive layout
- Grid-based product display
- Navbar and footer section

## 🧰 Tech Stack
- HTML5, CSS3
- Font Awesome Icons
